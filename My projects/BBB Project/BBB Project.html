Blood Bank Bahrain Project

Project Overview

This project is a comprehensive Blood Bank Management System designed to streamline the process of blood donation and distribution in Bahrain. The system consists of a website and a mobile application, both connected to the same database to ensure seamless data management and accessibility.

Web Platform

Full-featured website for donors, recipients, and administrators with real-time blood inventory management.

Mobile Application

Native mobile app for easy access to blood donation services and emergency requests.

Unified Database

Centralized database ensuring data consistency across both web and mobile platforms.
